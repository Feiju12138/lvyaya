# lvyaya

* 产品介绍：

  绿鸭鸭垃圾分类项目，由Sun Shine团队倾力打造，主要功能为：

  1. 通过输入垃圾名称，检索垃圾所属类别
  2. 通过输入所在城市，检索垃圾站位置

* 源代码使用说明：

  1. 下载源代码：

     ```bash
     cd ~/Downloads
     git clone -b codegen https://github.com/Feiju12138/lvyaya.git
     ```

     

  2. 编译：

     1. 前端

        ```shell
        cd ~/Downloads/lvyaya/lvyaya-fe/
        sudo npm install
        sudo npm start
        ```

     2. 后端

        ```bash
        cd ~/Downloads/lvyaya/
        
        ```
